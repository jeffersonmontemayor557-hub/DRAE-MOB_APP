Functional Coverage by Role

Resident / User
  • Sign up / login with email + password
  • Mandatory profile completion (when flagged)
  • View weather, advisories, guides, emergency hotlines
  • Submit incident report (text + photo/audio + location)
  • Offline queue retry and report history tracking
Staff
  • Authenticated via profile linked to staff_members
  • Receive assignment updates
  • Open staff assignment list
  • Update report progress and final resolution
  • Refresh assignment stats in real time
Admin (Dashboard)
  • Create/edit resident profiles and mobile logins
  • Create/edit staff and optional mobile login
  • Monitor reports and adjust assignment/status
  • Manage hotlines and advisories
  • Operational CRUD for key response data
Superadmin
  • Create admin authentication accounts
  • Grant admin or superadmin role
  • Enforce admin password change policy
  • Remove admin access
  • Own platform-level RBAC governance
